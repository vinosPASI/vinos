# Paleta de Colores – Vinoteca ML

Esta paleta de colores fue diseñada para transmitir una experiencia visual elegante, cálida y coherente con la temática del vino. Se basa en tonos pastel que mejoran la legibilidad, reducen la fatiga visual y aportan una estética moderna y premium tanto en la aplicación móvil como en la versión web.

## Colores Principales

### `#8C4A5A` – Vino Pastel
**Uso:**
* Botones principales
* Barra de navegación
* Elementos clave de la interfaz
* Branding (logo, títulos importantes)

### `#6E3B47` – Vino Oscuro Suave
**Uso:**
* Estados hover (al pasar el cursor)
* Botones activos
* Elementos seleccionados
* Encabezados o secciones destacadas

## Colores Secundarios

### `#F3E5AB` – Amarillo Vainilla
**Uso:**
* Fondos de secciones
* Tarjetas (cards)
* Formularios
* Áreas de contenido secundario

### `#E6D3A3` – Dorado Pastel
**Uso:**
* Íconos
* Líneas decorativas
* Bordes sutiles
* Detalles visuales (usar con moderación)

## Color Neutro

### `#F8F5F0` – Crema Claro
**Uso:**

* Fondo principal
* Pantallas generales
* Contenedores base
* Layout general

## Guías de Diseño

* Mantener consistencia en toda la aplicación.
* Usar los colores principales para jerarquía y acciones clave.
* Limitar el uso del color de acento para evitar sobrecarga visual.
* Garantizar buen contraste para la legibilidad del texto.
* Mantener una interfaz limpia, moderna y minimalista.
